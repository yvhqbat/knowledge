## 20.有效的括号
## 题目描述
```
给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串，判断字符串是否有效。

有效字符串需满足：

左括号必须用相同类型的右括号闭合。
左括号必须以正确的顺序闭合。
注意空字符串可被认为是有效字符串。

示例 1:

输入: "()"
输出: true
示例 2:

输入: "()[]{}"
输出: true
示例 3:

输入: "(]"
输出: false
示例 4:

输入: "([)]"
输出: false
示例 5:

输入: "{[]}"
输出: true

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/valid-parentheses
著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。
```

## 思路
使用map保存括号信息，使用stack作为辅助空间。

## 答案
cpp code:
```cpp
class Solution {
public:
    bool isValid(string s) {
        std::map<char,char> mapper;
        mapper['(']=')';
        mapper['{']='}';
        mapper['[']=']';

        std::stack<char> stacker;
        for(char c : s){
            if(mapper.count(c)>0){
                stacker.push(c);
            }else{
                if(stacker.empty()){
                    return false;
                }
                char top = stacker.top();
                if(mapper[top]==c){
                    stacker.pop();
                }else{
                    return false;
                }
            }
        }

        if(!stacker.empty()){
            return false;
        }
        return true;
    }
};
```

golang code:
```golang
func isValid(s string) bool {
    mapper := map[rune]rune{
        '{':'}',
        '[':']',
        '(':')',
    }

    stacker := make([]rune, len(s))
    top := -1
    for _, c := range s{
        if _,ok:=mapper[c];ok{
            top++
            stacker[top]=c
        }else{
            if top>=0 && mapper[stacker[top]]==c{
                top--
            }else{
                return false
            }
        }
    }
    if top>=0{
        return false
    }
    return true
}
```
